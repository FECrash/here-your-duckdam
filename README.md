## INTRO
```
여러분들은 평소에 명절마다 덕담을 보내야할 지 고민이 되시나요?
주변 사람들에게 특별한 덕담 메세지를 보내보는 건 어떠세요?

추석이라는 특별한 날을 빌려 비밀 덕담을 친구들에게 혹은 주변 지인들에게 선물해보세요! 
```
